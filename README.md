# ESG-BERT-analyzing-Reports

## 目標
在專案中我們應用ESG-BERT分析美國及台灣各25家頂尖公司在各自的年度ESG report 中關於Sustainable Investing中26個領域的貢獻。
透過NLP技術以及PDF汲取資訊，分析出不同公司側重的領域，以及台美差異。

## 結果
透過ESG-BERT輸出的label製成雷達圖比較，可看出與其他公司平均的差異，可能是該公司需要補強的部分
